Метод /server-add, где параметры передаютя напрямую через строку (имя пользователя, параметры серверной машины).
При GET запросе происходит поиск в базе данных по заданным параметрам. При POST запросе происходит добавления
пользователя с таким именнем, серверная машина с заданными параметрами и создается связь между пользователем и
этой машиной.

Метод /yaml-template, где параметры передаются через json файл в header запроса. поддерживает только метод POST.
При этом запросе шаблон json парсится и разбирается на компоненты, после чего происхожит добавление его структура в
базу данных и соотвественно ребер.

Не поддерживаются методы  jaml шаблоне get_inputs. Все параметры должны быть явно заданы.
